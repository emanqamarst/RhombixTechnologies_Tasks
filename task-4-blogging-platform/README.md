# The Daily Byte — Blogging Platform with Admin Panel

A sleek, fully responsive blogging platform built with pure HTML, CSS, and Vanilla JavaScript. Readers can browse, search, and comment on posts, while admins get a dedicated panel to manage content.


## Technologies Used

HTML5

CSS3

Vanilla JavaScript

LocalStorage API

Responsive Design

Editorial / Newsroom UI

Google Fonts (Fraunces, Inter & JetBrains Mono)


## Features

- ✅ Browse blog posts in a responsive card grid
- ✅ Search posts by title, content, or tags
- ✅ Filter posts by category
- ✅ Read full posts in a detail view
- ✅ Post comments as a reader (name + message)
- ✅ Admin login system (Press Room access)
- ✅ Create new blog posts
- ✅ Edit existing blog posts
- ✅ Delete blog posts
- ✅ Rich text editor (bold, italic, underline, headings, lists, links)
- ✅ Inline image upload inside post content
- ✅ Cover image upload for each post
- ✅ Category & tag assignment per post
- ✅ Comment moderation (view & delete any comment)
- ✅ Reader activity tracking (derived from comments)
- ✅ Remove all comments from a specific reader
- ✅ Toast/alert notifications for key actions
- ✅ LocalStorage data persistence
- ✅ Fully responsive (mobile, tablet, desktop)

## Admin Panel Overview

The Press Room (admin panel) is organized into three tabs:

- **Posts** — create, edit, and delete blog posts using the rich text editor
- **Comments** — moderate and remove any comment across the site
- **Readers** — view everyone who has commented and clear their activity

## Data Storage

The Daily Byte uses the LocalStorage API to store:

- Blog Posts
- Post Content, Tags & Categories
- Cover & Inline Images
- Comments & Commenter Names

No backend server or database is required.
