# Task 1 - To-Do List App

A simple To-Do List web app built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Tasks saved in Local Storage (persist after refresh)

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Simply open `index.html` in any browser.
