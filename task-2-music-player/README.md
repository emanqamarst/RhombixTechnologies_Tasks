# 🎵 Vibex — Premium Music Player

A stunning, fully responsive music player built with pure HTML, CSS, and Vanilla JavaScript.


## ⌨️ Keyboard Shortcuts

| Key         | Action           |
|-------------|------------------|
| `Space`     | Play / Pause     |
| `→`         | Next Song        |
| `←`         | Previous Song    |
| `↑`         | Volume Up +5%    |
| `↓`         | Volume Down -5%  |
| `M`         | Mute / Unmute    |
| `S`         | Toggle Shuffle   |
| `R`         | Toggle Repeat    |


## 🎨 Technologies Used

HTML5
CSS3
Vanilla JavaScript
LocalStorage API
Responsive Design
Glassmorphism UI

## 🎨 Features

- ✅ Play / Pause / Next / Prev / Shuffle / Repeat
- ✅ Volume slider + Mute toggle
- ✅ Animated progress bar with drag-to-seek
- ✅ Rotating album art while playing
- ✅ Equalizer bar + dot ring animations
- ✅ Floating particle background
- ✅ Sidebar playlist with live search
- ✅ Category filter (Pop, Lo-fi, Hip-Hop, Classical, Chill)
- ✅ Dark / Light mode toggle
- ✅ Toast notifications
- ✅ LocalStorage: saves volume, last song, theme, shuffle/repeat
- ✅ Fully responsive (mobile, tablet, desktop)
- ✅ Keyboard shortcuts
