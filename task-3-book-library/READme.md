#  LibraryOS — Personal Book Manager

A sleek, fully responsive personal library management application built with pure HTML, CSS, and Vanilla JavaScript.


## Technologies Used

HTML5

CSS3

Vanilla JavaScript

LocalStorage API

Responsive Design

Dark Theme UI

Google Fonts (Syne & Inter)


## Features

- ✅ Add new books to your library
- ✅ Edit existing book details
- ✅ Delete books from the collection
- ✅ Search books by title, author, or category
- ✅ Filter books by category
- ✅ Filter books by availability status
- ✅ Lend books to borrowers
- ✅ Return borrowed books
- ✅ Borrower name tracking
- ✅ Due date management
- ✅ Overdue book detection
- ✅ Complete borrowing history log
- ✅ Real-time library statistics
- ✅ Toast notifications for user actions
- ✅ Responsive sidebar navigation
- ✅ Modal-based forms
- ✅ LocalStorage data persistence
- ✅ Fully responsive (mobile, tablet, desktop)


## Book Categories

- Fiction
- Non-Fiction
- Science
- History
- Biography
- Self-Help
- Technology
- Philosophy


## Dashboard Statistics

The application automatically tracks:

- Total Books
- Borrowed Books
- Available Books
- Borrowing Activity

##  Data Storage

LibraryOS uses the LocalStorage API to store:

- Book Records
- Borrowing Information
- Due Dates
- Borrowing History

No backend server or database is required.
